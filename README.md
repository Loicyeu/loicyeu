##Fonctions
###Fonctions Locales

**Fonction testPassword() :**
- *ERR_PASSWORD_EMPTY* : Le mot de passe est vides
- *ERR_PASSWORD_INVALID* : Le mot de passe ne correspond pas aux critères exigés
- *ERR_PASSWORD_TOO_SHORT* : Le mot de passe est trop court
